# reflex-antd

A [Reflex](https://github.com/reflex-dev/reflex) custom component wrap [ant.design](https://ant.design/).

pypi: [reflex-antd](https://pypi.org/pypi/reflex-antd/)

## Installation

```bash
pip install reflex-antd
```
